# ml-create-app

## Install

```shell
npm i ml-create-app -g
```

## Usage

### To create app

```shell
ml-cli list // 查看模版列表

ml-cli init <template> <project> // 输入模版名称和项目名称
例如：ml-cli init template-react project-test
```

### Version

```shell
ml-cli --V
# or
ml-cli --version
```

### Help

```shell
ml-cli
# or
ml-cli -h
# or
ml-cli --help
```
